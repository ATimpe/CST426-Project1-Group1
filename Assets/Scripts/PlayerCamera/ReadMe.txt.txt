Add the PlayerCamera.cs script to a folder under Assets called Scripts.

Create a camera: call it PlayerCamera
	move camera to desired location: apply a 45 degree angle to x axis.


Apply a rigidbody component to PlayerCamera
	turn off gravity.

Apply the PlayerCamera.cs script to PlayerCamera as a component.

Under the PlayerCamera.cs script tab there is a little section called Rigidbody,
	click drag and pull the Rigidbody component to the little section.
